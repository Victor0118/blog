## AC代码

``` python
class Solution:
    def maxArea(self, height):
        """
        :type height: List[int]
        :rtype: int
        """
        a = 0 
        b = len(height) - 1
        maxWater = 0
        while a < b:
            w = min(height[a], height[b]) * (b - a)
            if w > maxWater:
                maxWater = w
            if height[a] < height[b]:
                a += 1
            else:
                b -= 1
        return maxWater
```
